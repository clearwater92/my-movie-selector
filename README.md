# 🎞My Movie Selector
## 설명
사용자가 키워드로 영화를 검색하면 TMDB(The Movie DataBase) API로부터 해당 데이터 리스트를 불러옵니다. 영화에 대한 상세 정보를 보고 흥미가 있으면 자신만의 리스트에 저장을 할 수 있는 애플리케이션입니다.
